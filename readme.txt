This project looks at how eating habits and physical conditions of people from Mexico, Peru and Colombia with agesbetween 14 and 61 can be used to estimate obesity levels.
The project uses data from UCI ML Repository. Various classification algorithms (KNN, Random Forest, LGBM) were implemented using Python Programming language and its packages and many performance metrics were applied to visualize and compare their performances and also finding the most important features.

The requirements.txt file contains the required version of the used packages. Any latest version of package listed there might work. 
The screen recording and the report is also included.
